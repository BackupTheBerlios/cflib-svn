Parameters:

stringError message format string. Giving it a NULL pointer results in an empty error message string
...Arguments list according to format string

Error Code and Message Inquiry Function.

cfgeterr() Error Code and Message Inquiry Function

Parameters:
stringPointer to a string, to which the error message should be copied. Giving it a NULL pointer will omit message return
lenSize of string, if 0 CF_MAXERRSTR will be used
Returns:
  • 0 : if no error is available, everything is alright
  • !=0 : error code of the next error in list
