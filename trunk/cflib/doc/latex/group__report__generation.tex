\hyperlink{group__report__generation_ga2695585ee0a7485da6075848fa27a33f}{cfform()} Process a Template from file or {\ttfamily stdin} and write generated Report to File or {\ttfamily stdout} 

Process a Template from file or {\ttfamily stdin} and write generated Report to File or {\ttfamily stdout}.

The Report Template contains {\bfseries Variables} like '\$({\itshape ident\/})' or whatever you set in {\itshape vd\/}. {\itshape ident\/} may refer to a\-:


\begin{DoxyItemize}
\item Parameter Name like '\$({\itshape name\/})' \-: Substitute variable by parameter's value (from D\-B)
\begin{DoxyItemize}
\item Include File Path like '\$(F\-I\-L\-E\-:{\itshape /my/path/to/incfile\/})' \-: Substitute variable by content of include file
\item File Path and Section Include like '\$(F\-I\-L\-E\-:{\itshape incfile\#{\itshape sect\/})'\/} \-: Include section from include file between \mbox{[}{\itshape sect\/}\mbox{]} and next \mbox{[}{\itshape \/}...\mbox{]}
\item Variable File Include like '\$(F\-I\-L\-E\-:\-:{\itshape varname\/})' \-: Include file from location given in parameter {\itshape varname\/} 
\end{DoxyItemize}
\end{DoxyItemize}
